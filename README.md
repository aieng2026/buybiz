# buybiz — Business Acquisition Toolkit

A working toolkit for an individual acquirer buying one established, owner-independent business. Built around a "Buy Then Build" thesis: acquire a business with a solid foundation and recurring revenue, then improve operations.

## Buy Box
- **EBITDA / SDE:** $600K+
- **Deal size:** up to $10M (SBA 7(a) + 504 combined)
- **Financing:** SBA, ~10% equity injection, seller note acceptable
- **Revenue:** recurring / contract-based
- **Management:** GM minimum in place (operate remotely / semi-absentee)
- **Geography:** FL, NJ, NY, PA, CT
- **Seller:** retiring, motivated
- **Health:** stable or growing (not turnaround)

## Target Industries (shortlist)
Commercial cleaning · IT managed services (MSP) · commercial landscaping · self-storage · RV/boat storage · car wash (express tunnel)

## Files

| File | What it is |
|---|---|
| `Acquisition_Screening_Scorecard.xlsx` | Interactive workbook to triage deals and vet brokers. Tabs: **Scorecard** (15-min deal triage, auto-verdict), **Broker Vetting** (rate brokers), **Scoring Guide**, **Industry Checks**, **Buy Box** |
| `Buyer_Introduction_Email.md` | Broker outreach templates (cold, warm intro, follow-up) + operating-background positioning |

## How to use
1. Send the intro emails to 5–8 brokers across target states/industries.
2. Score each broker on the **Broker Vetting** tab after the first call.
3. Run every live listing through the **Scorecard** tab — advance / clarify / pass.
